# Backend Server

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure `.env` file with your database credentials:
```
DB_HOST=localhost
DB_USER=your_user
DB_PASSWORD=your_password
DB_NAME=your_database
DB_PORT=3306
PORT=3000
```

3. Start the server:
```bash
npm start
```

Server runs on `http://localhost:3000`
